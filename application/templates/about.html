About this Web App

This application is to demonestrate and ability of the cohort to create a CURD Application

This application will enable it's users to follow the statistics of their favourite players. The user will need to follow this flow to be able to add layer statistics:

  1. Create an account with his/her name email and password
  2. Add a player to start tracking his statistics for the season
  3. Add stats to your added players ( can’t jump here without having at least one player to assign stats to )
    * * A player may have many stats on different dates.
  4. The user may input some data wrong and may want to edit them
  5. The user may decide he doesn’t want to keep track of one of the players that he already has therefore he should be able to delete players.
